# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be patient, assume good intent, and disagree without making it personal. Anyone interacting with the project — issues, PRs, discussions — is expected to follow it.

Reports of unacceptable behavior can go to the maintainer (see [SECURITY.md](SECURITY.md) for contact). All reports stay confidential.
