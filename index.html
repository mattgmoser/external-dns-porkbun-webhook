ExternalDNS Porkbun webhook Helm repository

This chart installs the official ExternalDNS controller and the Porkbun provider in one Pod, with the provider API restricted to loopback. 

helm repo add edns-porkbun https://mattgmoser.github.io/external-dns-porkbun-webhook
helm repo update
helm show values edns-porkbun/external-dns-porkbun-webhook > external-dns-porkbun-values.yaml
# Edit every example domain, Secret, and TXT ownership value.
helm upgrade --install external-dns edns-porkbun/external-dns-porkbun-webhook \
  --namespace external-dns --create-namespace \
  --values external-dns-porkbun-values.yaml

Read the complete prerequisites and migration guidance in the project README, or view the package on Artifact Hub.
